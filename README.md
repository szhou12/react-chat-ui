# React Chat UI

## Setup
```bash
npm create vite@latest frontend --template react
 > Select a framework: React
 > Select a variant: JavaScript

cd frontend

npm install

npm install @chatscope/chat-ui-kit-react

npm install @chatscope/chat-ui-kit-styles
```
